Měření A0 / C180

Zatím nejsou zadané hodnoty.

Výsledek po 1. měření – zapsat před měřením A osy

Vzorec
7530.0 nový = 7530.0 původní + X / 2 7530.1 nový = 7530.1 původní + Y / 2 7530.2, 7530.4, 7530.5 = beze změny Kontrola: pokud ABS(Z) > 0.015, Excel upozorní na nastavení A osy / MP960.

Výsledek po 2. měření – finální hodnoty k zapsání

Vzorec
průměr = (Y + Z) / 2 7530.4 nový = 7530.4 původní + (Y - průměr) 7530.5 nový = 7530.5 původní + průměr 7530.0, 7530.1, 7530.2 = beze změny Kontrola: pokud ABS(X) > 0.020, Excel upozorní na kontrolu geometrie Z osy v X směru.
